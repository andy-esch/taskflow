# Fit 1x drivetrain

## Objective

Simplify to a 1x setup for fewer things to go wrong mid-race. Remove the front
derailleur and shifter, fit a narrow-wide ring and a clutch rear derailleur, and
set the chain length for the new range.

## Acceptance criteria

- [x] Front derailleur and left shifter removed
- [x] Narrow-wide chainring installed
- [x] Clutch rear derailleur fitted and indexed
- [x] Chain sized and retention tested over rough ground

## Out of scope

- A wider-range cassette — the current spread is enough.
