# Replace chain and cassette

## Objective

The chain measured past 0.75% and the cassette had begun to hook, so shifts
skipped under climbing load. Replace both as a set and re-index so the drivetrain
runs quiet and holds every gear fully loaded.

## Acceptance criteria

- [x] Worn chain removed and drivetrain degreased
- [x] New cassette and chain fitted, sized to big-big plus two links
- [x] Indexing dialed across the full range
- [x] Test climb: no skip under load

## Out of scope

- Chainring replacement — still within wear.
