# Install dropper post

## Objective

Add a dropper to get the saddle out of the way on loose descents. Route the cable
internally, set the lever at the bar, and dial the return speed.

## Acceptance criteria

- [ ] Post fitted and cable routed internally
- [ ] Lever mounted and throw set
- [ ] Return speed and travel dialed

## Out of scope

- Frame re-routing — using the existing ports.
