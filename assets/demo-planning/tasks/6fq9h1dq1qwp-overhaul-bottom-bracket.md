# Overhaul bottom bracket

## Objective

A creak tracks pedal pressure out of the saddle. Pull the crank, isolate whether
it's the BB cups, the crank interface, or the pedals, then clean, grease, and
re-torque — replacing the BB only if it's actually notchy.

## Acceptance criteria

- [x] Crank removed, interfaces degreased
- [ ] Creak isolated (BB vs crank vs pedals)
- [ ] Interfaces greased and torqued to spec
- [ ] Test climb: silent under load

## Out of scope

- Switching BB standards — staying with the current shell.
