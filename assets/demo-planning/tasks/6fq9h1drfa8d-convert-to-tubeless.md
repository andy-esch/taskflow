# Convert to tubeless

## Objective

Drop the tubes to run lower pressures for grip and fewer pinch flats on chunky
gravel. Tape the rims, seat tubeless tires, add sealant, and confirm they hold
overnight.

## Acceptance criteria

- [x] Rims taped and valves fitted
- [x] Tires seated and beads popped
- [x] Sealant added and spun to coat
- [x] Held pressure overnight

## Out of scope

- New rims — the current ones are tubeless-ready.
