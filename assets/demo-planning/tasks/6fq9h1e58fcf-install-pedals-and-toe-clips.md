# Install pedals and toe clips

## Objective

Set up the contact points on the fixed-gear: grease and torque the pedals to the
right cranks, then fit toe clips and straps sized for daily commuting shoes.

## Acceptance criteria

- [ ] Pedals greased and torqued (mind the left reverse thread)
- [ ] Toe clips and straps fitted
- [ ] Clip size and strap tension set for commute shoes

## Out of scope

- Clipless pedals — running clips and straps by choice.
